Rainy Days

This is my first Front-end project called "Rainy Days." This project is a simple online clothing store that I developed as part of my Front-End Development studies at Noroff School. HTML and CSS were used to build the website.

Rainy Days sells men's and women's rain jackets, with a target audience aged 30 to 50, interested in outdoor activities like hiking, skiing, and more.

Community Science Museum

This website is my submission for Semester Project 1 in my studies at Noroff. This project is a modern and responsive web presence for an interactive museum called the "Community Science Museum," built using HTML and CSS.

The target audience is primary and middle school children and families. The main goal is to excite people enough to come visit the museum.
